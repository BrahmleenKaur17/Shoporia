# 🛒 Shoporia – MERN E-commerce App

## 🚀 Overview

Shoporia is a full-stack MERN e-commerce web application with an integrated AI voice assistant for navigation.

## ✨ Features

* 🛍️ Product browsing & filtering
* 🛒 Cart & order management
* 🔐 User authentication
* 💳 Payment integration (Razorpay)
* 🎤 AI voice-controlled navigation

## 🛠️ Tech Stack

* Frontend: React, Vite, CSS
* Backend: Node.js, Express
* Database: MongoDB
* Other: Razorpay API, Speech Recognition

## 📌 How to Run

```bash
# frontend
cd frontend
npm install
npm run dev

# backend
cd backend
npm install
npm start
```
